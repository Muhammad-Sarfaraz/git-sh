# git-sh

A simple shell script utility to automate Git repository creation and setup.

## Overview

`git-sh` is a lightweight helper script that simplifies the process of creating new Git repositories and initializing them quickly from the terminal.

It reduces repetitive commands and speeds up project setup.

## Features

- Create new Git repositories instantly
- Initialize local projects with Git
- Automate common Git setup steps
- Lightweight shell-based utility

## Usage

Run the script from terminal:

```bash
sh git.sh
```

or

```bash
./git.sh
```

## What it does

Typically automates:

- `git init`
- `git add .`
- `git commit`
- remote repository setup (optional)

## Purpose

This tool is designed to:

- Save time during project initialization
- Reduce manual Git setup steps
- Help developers quickly start new repositories
